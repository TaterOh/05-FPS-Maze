# 05-FPS-Maze
FPS Maze project for MSCH-C220 finished 10 April 2022.

Moves with WASD and shoot with the spacebar or leftclick.

## Implementation

An enemy

A gun

Music

Sound effects

Procedurally generating maze

Pause, win, & lose screens

Occlusion for windows
